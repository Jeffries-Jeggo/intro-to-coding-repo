# Intro to Coding - Python Exercises

A set of 40 exercises for Year 9 students learning Python, organized into 8 lessons.

## Structure

```
exercises/
  1-Functions-and-Arithmetic-Operators/
    1.1/, 1.2/, 1.3/, 1.4/, 1.5/
  2-Variables-and-Assignment/
    2.1/, 2.2/, ...
  ...
  8-Strings/
tests/
  1.1/, 1.2/, ... (one test file per exercise)
teacher/
  markscheme/
    1.1/, 1.2/, ... (solutions - teacher only)
```

## How to Use

1. Open this repo in GitHub Codespaces
2. Navigate to your current lesson's exercises folder
3. Read the README.md in each exercise folder
4. Write your code in student.py
5. Run your code: python student.py
6. Run tests: pytest tests/X.X/test_X_X.py -v
