## Objectives
Use Python's arithmetic operators: +, -, *, /, **, %

## Task
Python supports all the standard math operators:
- +  addition
- -  subtraction
- *  multiplication
- /  division
- ** power (exponent)
- %  modulo (remainder)

Example:
print(5 + 3)   # prints 8
print(10 - 4)  # prints 6
print(3 * 7)   # prints 21
print(20 / 4)  # prints 5.0
print(2 ** 4)  # prints 16 (2 to the power of 4)
print(17 % 5)  # prints 2 (remainder when 17 divided by 5)
