# Exercise 1.4: Order of Operations

## Objectives
Understand that Python follows standard order of operations (PEMDAS).

## Task
Python follows standard math order of operations:
P - Parentheses first
E - Exponents (power)
M/D - Multiply/Divide left to right
A/S - Add/Subtract left to right

When functions are involved, work from the inside out.

Example:
max(5 + 3, 10 - 2) = max(8, 8) = 8

Your Task - calculate and print:
1. max(4 + 1, 6 - 1)
2. min(10 + 2, 15 - 3)
3. abs(5 - 20)
4. int(7.5 + 2.3)
5. max(3 * 2, 20 / 2)
6. min(10 * 2, 50 / 2)
